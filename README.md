# Weather App for iOS

Weather app provides a forecast for the upcoming 16 days. Based on HTTP Requests

# Features

- Table and Cells
- Alamofire
- [OpenWeatherMap](http://openweathermap.org)

# Version
Developed and Tested on iOS 10, Xcode 8 and Swift 3
